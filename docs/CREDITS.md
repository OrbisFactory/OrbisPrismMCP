# Third-Party Software Credits and Attribution

Orbis Prism MCP leverages several open-source tools to facilitate reverse engineering and modding for Hytale. We are grateful to the authors of these projects for their incredible work.

## Decompiler Engines

### [JADX](https://github.com/skylot/jadx)
- **Description**: Dex to Java decompiler. Used for fast and high-fidelity decompilation.
- **License**: [Apache License 2.0](http://www.apache.org/licenses/LICENSE-2.0)
- **Copyright**: Copyright 2018 by Skylot

### [Vineflower](https://github.com/Vineflower/vineflower)
- **Description**: The most technically advanced Java decompiler, a fork of Quiltflower/Fernflower.
- **License**: [Apache License 2.0](http://www.apache.org/licenses/LICENSE-2.0)
- **Copyright**: Copyright 2024 by various contributors

## Other Libraries and Tools
- **Rich**: For terminal user interface and elegant progress bars.
- **Typer**: For creating the command-line interface (CLI).
- **SQLite (FTS5)**: Full-text search engine for the indexed API.
